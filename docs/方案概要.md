# GainLab 方案概要

> **一句话**：金融衍生品数据 + 可视化组件 + API，给交易者看图、给开发者嵌入、给 AI Agent 调用。
>
> 类比：TradingView Widget + CoinGlass API 的合体，专为 AI Agent 时代设计。

---

## 为什么做这个

### 市场缺什么

| 现有产品 | 它能干什么 | 它不能干什么 |
|----------|-----------|-------------|
| TradingView | K 线图表，90M 用户 | 没有 Perp DEX 数据，没有 Agent 接口 |
| CoinGlass | CEX 衍生品数据 | 图表不能嵌入，没有 Agent 接口 |
| DexScreener | DEX 现货数据 | 没有衍生品数据 |
| FMP | 美股/外汇数据 | 没有加密衍生品 |

**空白地带**：没有一个产品能同时做到——聚合衍生品数据 + 可嵌入图表 + AI Agent 可调用。

### 市场规模

- 加密衍生品市场：**$85.7T**（2025）
- Perp DEX 交易量：**$7.348T**（2025，同比 +346%）——增长最快，数据覆盖最差
- TradingView 收入：**$250M+**（2025），估值 $3B
- DexScreener 收入：**$125M**（纯广告模式）
- CoinGlass API 定价：**$29-699/月**

## 做什么

### 三层架构

```
┌─────────────────────────────────────────────┐
│  Layer 3 — 变现                              │
│  Agent 接口：REST API / MCP / OpenAI Function │
│  SDK：@gainlab/sdk (JS) / gainlab (Python)   │
├─────────────────────────────────────────────┤
│  Layer 2 — 差异化                            │
│  可嵌入组件：iframe / React / Vue            │
│  K线、热力图、清算瀑布、OI 仪表板             │
├─────────────────────────────────────────────┤
│  Layer 1 — 数据护城河                        │
│  CEX + Perp DEX 衍生品数据聚合               │
│  资金费率 / 持仓量 / 清算 / 基差 / 订单流     │
└─────────────────────────────────────────────┘
```

### 核心产品

| 产品形态 | 说明 | 用户 |
|----------|------|------|
| **gainlab.ai 网站** | 直接看图表，免费 | 散户交易者 |
| **可嵌入 Widget** | 一行代码贴到任何网站 | 开发者、社区站长 |
| **REST API** | JSON 数据接口 | 量化开发者 |
| **MCP Server** | AI Agent 工具调用 | Claude/OpenClaw/ChatGPT 等 Agent |
| **可分享图表链接** | gainlab.ai/chart/xxx | KOL、分析师 |

### 六个核心组件

1. **K-Line Pro** — 多周期 K 线，45+ 指标（KLineChart 底座）
2. **Funding Heatmap** — 多交易所资金费率对比热力图
3. **Liquidation Waterfall** — 清算水位瀑布图
4. **Open Interest Dashboard** — 持仓量 + 多空比 + 巨鲸追踪
5. **Basis Spread Monitor** — 现货-期货基差 + 套利机会
6. **Market Flow** — 订单流 + 买卖压力

## 差异化

1. **Perp DEX 数据**：$7.348T 市场，CoinGlass 和 TradingView 都不覆盖。Hyperliquid、dYdX、Drift、GMX 链上数据。
2. **可嵌入**：CoinGlass 图表不能嵌入别人网站，GainLab 可以。跟 YouTube 嵌入视频一样简单。
3. **Agent 可调用**：TradingView 和 CoinGlass 都没有 MCP Server / Function Calling 接口。GainLab 有。
4. **多协议支持**：一个核心 API，同时提供 REST / MCP / OpenAI Function / LangChain 四种接入方式。

## 技术方案

### 不造轮子，用现成的组装

| 组件 | 用什么 | 成本 |
|------|--------|------|
| K 线渲染 | KLineChart（已有） | 免费 |
| CEX 衍生品数据 | CoinGlass API | $29-699/月 |
| Perp DEX 数据 | Hyperliquid/dYdX/Drift 链上 API | 免费 |
| 美股/外汇 | FMP（已有订阅） | $8/月 |
| 多市场补充 | Twelve Data / Binance 公开 API | 免费-$79/月 |
| 前端框架 | React / Next.js | 免费 |
| 部署 | Vercel / Cloudflare | 免费起步 |

### Agent 接口架构

```
                     ┌── MCP Server (Claude/Cursor/OpenClaw)
                     │
GainLab REST API ───├── OpenAI Function Schema (ChatGPT/GPTs)
                     │
                     ├── LangChain Tool Wrapper
                     │
                     └── JS/Python SDK (@gainlab/sdk)
```

核心是 REST API，其他都是薄包装。不赌单一协议。

## 与 OpenClaw 的关系

```
用户 ←→ OpenClaw（AI Agent 交互）←→ GainLab（数据 + 可视化）
```

- **OpenClaw** = 水龙头（用户界面、对话、分析、推送）
- **GainLab** = 水管和仪表盘（数据、图表、API）
- 互补不竞争。OpenClaw 调 GainLab 的 MCP/API，拿数据、生成图表给用户看。

## 风险与应对

| 风险 | 严重度 | 应对 |
|------|--------|------|
| CoinGlass 做 Widget 功能 | ⚠️ 高 | 抢先做 Perp DEX 数据（他们不覆盖）+ Agent 接口 |
| 一个人维护 50+ 交易所 | ⚠️ 高 | MVP 先做 5-8 个主流交易所，不贪多 |
| MCP 生态没起来 | ⚠️ 中 | 核心做 REST API，MCP 只是包装层 |
| 免费模式需要大流量 | ⚠️ 中 | 先靠 API/Widget 付费，广告是锦上添花 |
| 大厂直接做 | ⚠️ 中 | 垂直领域（金融衍生品）+ 快速迭代是壁垒 |
